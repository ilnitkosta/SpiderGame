### Save the spider Game
Learning project using Pygame.

## How to run it locally

# 1. Install git
First, sure git is installed on your computer:
``` bash []
git --version
```
If the output is empty, do next

On Ubuntu/Debian:
``` bash []
sudo apt update
sudo apt install git
```
On macOS:
``` bash []
brew install git
```
# 2. Clone the repo
Use the git clone command with the repository’s URL:
``` bash []
git clone https://github.com/ilnitkosta/SpiderGame
```

# 3. Navigate to the Game Folder
After cloning, navigate to the game’s directory:
``` bash []
cd SpiderGame
```

# 4. Launch the Game

Run the game using the command:
``` bash[]
python3 game.py
```

Enjoy!
